## Strategische Fragen

### Was wird Labrechner langfristig?

**Option A:** Fokussierte BEL-Suche + KI-Chat
- Pro: Klarer Fokus, schnelle Iteration
- Con: Begrenztes Upselling-Potenzial

**Option B:** Vollständiges Labor-Management
- Pro: Höherer Customer Lifetime Value
- Con: Starke Konkurrenz, langer Entwicklungszyklus

→ Entscheidung nach V2-Launch basierend auf Nutzerfeedback
